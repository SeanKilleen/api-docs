Style

You can rely on the base styles in power-up.css. There’s no need to override things like buttons, inputs, tables, headers, font styles, and code blocks, that aren’t core to your Power-Up. power-up.css includes normalize.css, a CSS reset which “makes browsers render all elements more consistently and in line with modern standards”. You don’t need to include anything similar in your CSS.
