session

GET /1/sessions/socket

  • Notes: This is the route for WebSocket requests. See the socket API reference for a description of WebSocket usage.
  • Required permissions: read
  • Arguments: None

PUT /1/sessions/[idSession]

  • Required permissions: write
  • Arguments
    • status (required)
      • Valid Values: One of:
        • active
        • disconnected
        • idle
    • idSession (required)
      • Valid Values: a string with a length from 16 to 24
    • idBoard (optional)
      • Valid Values: The id of the board you're viewing. Boards with no viewers will not get updates about members' statuses.

PUT /1/sessions/[idSession]/status

  • Required permissions: write
  • Arguments
    • value (required)
      • Valid Values: One of:
        • active
        • disconnected
        • idle
    • idSession (required)
      • Valid Values: a string with a length from 16 to 24

POST /1/sessions

  • Required permissions: write
  • Arguments
    • status (required)
      • Valid Values: One of:
        • active
        • disconnected
        • idle
    • idBoard (optional)
      • Valid Values: The id of the board you're viewing. Boards with no viewers will not get updates about members' statuses.
